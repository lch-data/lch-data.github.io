## 🎨 Theme & Design

이 포트폴리오는 **GitHub Pages + Cayman Theme** 기반으로 제작되었습니다.

### 왜 Cayman 계열 테마를 선택했나요?

- 직장과 학업을 병행하는 상황에서  
  **과한 커스터마이징보다는 유지 보수가 쉬운 심플한 구조**를 우선했습니다.
- 텍스트 가독성이 좋고,  
  **콘텐츠 중심**으로 정보를 보여주기에 적합했습니다.
- 불필요하게 화려하지 않고,  
  **깔끔한 UI로 경력/프로젝트를 명확하게 전달**할 수 있다는 점이 장점입니다.

기본 Cayman 테마 위에  
- 섹션 구성 재배치  
- 일부 색상 및 레이아웃 수정  
- 이미지/영상 배치 레이아웃 조정  
등을 통해 **저만의 포트폴리오 느낌**을 줄 수 있도록 커스터마이징했습니다.

## 🧱 Structure & Contents (구성)

포트폴리오는 크게 아래와 같은 섹션으로 구성됩니다.

1. **Intro / Profile**
   - 이름, 현재 역할, 간단한 자기소개
   - 프랜차이즈 운영관리와 AI 학습을 병행하고 있다는 점 강조

2. **Experience (경력)**
   - 카페·브런치 매장 운영
   - 방탈출/체험형 콘텐츠 매장 관리
   - 프랜차이즈 본사 슈퍼바이저 및 운영팀장 경험
   - 각 경력마다
     - 담당 역할
     - 매출/운영 개선 성과
     - 인원 관리, QSC 관리 등 핵심 포인트를 정리

3. **Projects (프로젝트)**
   - 인공지능 및 데이터 관련 과제/프로젝트
   - 프랜차이즈 운영과 연결되는 분석/자동화 아이디어
   - 주요 내용:
     - 어떤 문제를 해결하려고 했는지(Problem)
     - 어떤 방법/도구를 사용했는지(Approach)
     - 결과 및 배운 점(Result & Learning)

4. **Skills (기술 스택)**
   - Language: Python 등
   - Tools: Git, GitHub, Colab, Jupyter 등
   - AI/ML: 기본적인 모델링·전처리·평가지표 이해
   - Operations: 매장 운영, 인력 관리, 고객 경험 설계 등

5. **Gallery (이미지/영상 섹션)**
   - 실제 매장/체험형 콘텐츠 사진 혹은 콘셉트 이미지
   - 프로젝트 혹은 포트폴리오 화면 캡처
   - 간단한 설명과 함께 배치하여 **시각적으로 이해하기 쉽도록 구성**

## ⚙️ 사용 기술 및 구현 포인트

### 1. GitHub Pages & Cayman Theme
- `pages-themes/cayman` 기반 포트폴리오 사이트
- `config` 파일 및 `README`/마크다운 문서를 이용한 정적 페이지 구성

### 2. Generative AI 활용
- 11주차에 미리 설계한 전체 구성을 바탕으로  
  생성형 AI를 활용해 **기본 레이아웃과 문서 구조를 자동 생성**하고,  
  이후 수동으로 내용을 다듬는 방식으로 작업 효율을 높였습니다.
- 이미지/영상 배치, 섹션 문구 등을 AI와 함께 계속 수정하며  
  **반복적으로 개선**했습니다.

### 3. Dynamic Features (동적 기능)
- 12주차에 수업에서 배운 내용을 바탕으로  
  **동적 요소(간단한 애니메이션, 스크롤 효과 등)**를 검색·적용했습니다.
- 정적 페이지이지만, 사용자가 지루하지 않게  
  **시선이 자연스럽게 흐르도록 하는 UI/UX**를 의도했습니다.
