# Cayman 포트폴리오 최종 세트

이 폴더의 내용은 GitHub Pages용 완전한 Cayman 테마 기반 포트폴리오입니다.

### 구성 파일
- `_config.yml` → 내장 테마(jekyll-theme-cayman) 설정
- `index.md` → 메인 포트폴리오 페이지 (About, Experience, Metrics 등)

### 업로드 방법
1. `lch-data.github.io` 저장소에 업로드
2. Settings → Pages → Branch: main / Folder: /(root)
3. https://lch-data.github.io 접속 후 스타일 적용 확인
